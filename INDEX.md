## 🚀 БЫСТРЫЙ СТАРТ

### КОПИПАСТА для PythonAnywhere Bash консоли:

```bash
# 1. Создание папки и загрузка файлов (уже должны быть!)
cd ~/telegram_bot

# 2. Создание окружения
mkvirtualenv --python=/usr/bin/python3.10 telegram_bot
workon telegram_bot

# 3. Установка пакетов
pip install -r requirements.txt

# 4. Установка токена
echo "export TELEGRAM_BOT_TOKEN='твой_токен'" >> ~/.bashrc
source ~/.bashrc

# 5. Проверка
python bot.py

# 6. Ctrl+C чтобы остановить
```

Потом в веб-интерфейсе:
- Tasks → Create new always-on task
- Вставить: `/home/username/.virtualenvs/telegram_bot/bin/python /home/username/telegram_bot/bot.py`
- Create
- ГОТОВО! 🟢
